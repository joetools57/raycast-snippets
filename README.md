# Raycast Snippets

A community repository for sharing Raycast text snippets, templates, and examples.

Repository layout
- /snippets/ — individual snippet files (Markdown or JSON)
- /examples/ — example workflows or longer templates
- CONTRIBUTING.md — contribution guidelines
- LICENSE — license for snippet content

How to use
- Browse or clone the repository and copy snippets into your Raycast Snippets.
- To contribute: fork, create a branch, add your snippet under `snippets/`, and open a pull request.

Snippet format suggestion (Markdown frontmatter)
---
title: Example Snippet
tags: [template, email]
language: text
author: joetools57
license: CC0
---
Snippet body goes here.

License
Choose a license for the snippet content. For code, use MIT; for text/snippets consider CC0 or CC-BY. See LICENSE file.
